### API Documentation Generator

**Prompt Title**: "REST API Documentation"

**Usage**: Type `/api` when documenting APIs

**Content**:
```
Generate comprehensive API documentation for:

**API Overview:**
- Name: [API NAME]
- Version: [VERSION NUMBER]
- Base URL: [BASE_URL]
- Authentication: [AUTH METHOD]

**Endpoint Documentation:**

### [HTTP_METHOD] /[endpoint-path]

**Description:** [WHAT THIS ENDPOINT DOES]

**Authentication Required:** [YES/NO]

**Request Parameters:**

**Path Parameters:**
- `[param_name]` (type, required/optional): [DESCRIPTION]

**Query Parameters:**
- `[param_name]` (type, required/optional): [DESCRIPTION]
- `[param_name]` (type, required/optional): [DESCRIPTION]

**Request Headers:**
```
Content-Type: application/json
Authorization: Bearer [TOKEN]
[OTHER_HEADERS]
```

**Request Body Example:**
```json
{
  "[field_name]": "[example_value]",
  "[field_name]": "[example_value]"
}
```

**Response Examples:**

**Success Response (200 OK):**
```json
{
  "status": "success",
  "data": {
    "[field_name]": "[value]",
    "[field_name]": "[value]"
  },
  "message": "[SUCCESS_MESSAGE]"
}
```

**Error Responses:**

**400 Bad Request:**
```json
{
  "status": "error",
  "error_code": "INVALID_REQUEST",
  "message": "[ERROR_DESCRIPTION]",
  "details": {
    "field": "[FIELD_NAME]",
    "issue": "[SPECIFIC_PROBLEM]"
  }
}
```

**401 Unauthorized:**
```json
{
  "status": "error",
  "error_code": "UNAUTHORIZED",
  "message": "Invalid or missing authentication token"
}
```

**Rate Limiting:**
- Limit: [REQUESTS_PER_MINUTE]
- Headers: `X-RateLimit-Remaining`, `X-RateLimit-Reset`

**Code Examples:**

**cURL:**
```bash
curl -X [METHOD] "[FULL_URL]" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer [TOKEN]" \
  -d '[REQUEST_BODY]'
```

**JavaScript (fetch):**
```javascript
fetch('[FULL_URL]', {
  method: '[METHOD]',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer [TOKEN]'
  },
  body: JSON.stringify([REQUEST_BODY])
})
.then(response => response.json())
.then(data => console.log(data));
```

**Python (requests):**
```python
import requests

response = requests.[method](
    '[FULL_URL]',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer [TOKEN]'
    },
    json=[REQUEST_BODY]
)
print(response.json())
```

Please customize this documentation for: [SPECIFIC_API_ENDPOINT]
```
